# ♟️ Chess Board

A visually styled chess board built using **HTML** and **CSS**, displaying a complete standard chess setup with all pieces in their starting positions.

---

## 📋 Overview

This project renders a static 8×8 chess board using an HTML `<table>`, with alternating dark and light squares. All 32 chess pieces (16 per side) are displayed using SVG images sourced from Wikimedia Commons.

---

## 🎨 Features

- Full 8×8 chess board with correct alternating square colors
- All standard chess pieces placed in their initial positions:
  - ♜ Black pieces on rows 1–2
  - ♙ White pieces on rows 7–8
- Chess piece images loaded from **Wikimedia Commons** (SVG format)
- Simple and clean layout with a **dark sea green** background
- Responsive table-based board structure

---

## 🛠️ Technologies Used

| Technology | Usage |
|---|---|
| HTML5 | Board structure using `<table>` |
| CSS3 | Styling squares, layout, and colors |
| SVG Images | Chess piece icons via Wikimedia Commons |

## 🚀 How to Run

2. Open `chess_board.html` in any modern web browser
3. No dependencies, no installation required!
